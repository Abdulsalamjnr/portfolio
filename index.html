Project 1
BASIC CALCULATOR

A calculator built with HTML, CSS, and JavaScript is a web-based application that allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. The core functionality of the calculator is implemented using JavaScript. Users can input numerical values and perform calculations by clicking on buttons representing digits (0-9) and mathematical operators (+, -, *, /). As users input their calculations, JavaScript dynamically evaluates the expressions and displays the results in real-time on the calculator's display screen.

Project 2
WEIGHT CONVERTER

The main functionality of the weight converter is implemented using Html, CSS and JavaScript. Users can input a weight value in one unit (such as kilograms, pounds, or grams) into a text field. As they type, JavaScript dynamically converts this value into the other supported units, displaying the results instantly on the webpage. This allows users to see the equivalent weight in multiple units simultaneously.

PROJECT 3
Leaps Year Checker

Leap year checker built using HTML, CSS, and JavaScript, The HTML file provides the structure of the webpage. It includes input fields for the user to enter a year and a button to trigger the leap year check. The CSS file styles the HTML elements, providing visual appeal and layout to the webpage while the JavaScript file contains the logic for checking whether the entered year is a leap year or not. It typically involves retrieving the input value, performing the leap year calculation (using conditions such as divisible by 4 but not by 100 unless also divisible by 400), and displaying the result on the webpage.
